## Logo

### The Mark
The LESARUSS logo combines a light bulb and a microphone into a single icon — ideas meet voice. The light bulb represents insight, clarity, and illumination. The microphone inside it represents communication, broadcast, and amplification. Together they communicate what LESARUSS does: help operators build and amplify systems that actually work.

### Wordmark
**"LESARUSS"** is set in ultra bold, condensed, all-caps sans-serif — heavy, confident, zero ambiguity.
**"PROJECT"** is set in a light/thin weight of the same sans-serif family with wide letter-spacing — creating visual contrast and balance without introducing a second typeface.

### Logo Versions
The following versions are confirmed from brand assets:

**Horizontal Lockup (Light Background)**
1. **Primary** — Full color. Orange bulb with black outline details, black "LESARUSS" wordmark, gray "PROJECT." Use on white or light backgrounds.
2. **Secondary** — Same as primary at a slightly different size rendering.
3. **Compact** — Tighter horizontal lockup for smaller applications (headers, footers, thumbnails).

**Reversed / White Wordmark (Dark Background)**
4. **Full Reversed** — Orange icon + white wordmark. For use on black, dark, or brand-color backgrounds. (Confirmed from assets — wordmark is fully white in this version.)
5. **Compact Reversed** — Smaller reversed lockup.

**Icon Only**
6. **Icon Large** — Bulb + mic mark only, no wordmark. Orange with black details. Use for profile images, favicons, stickers, and applications below 120px.
7. **Icon Small** — Same mark at reduced size.

**Versions still to create (recommended):**
- Stacked lockup (icon above wordmark, for square formats)
- All-black single-color version (for embroidery, stamps, single-color print)
- All-white single-color version (for dark print applications)

### Logo Usage Rules
- Do not alter the proportions between the icon and the wordmark
- Do not recolor the icon outside of brand-approved color versions
- Do not place the logo on backgrounds that reduce contrast below legible threshold
- Maintain clear space of at least the height of the "L" in LESARUSS on all sides
- Minimum size: do not use the full lockup below 120px wide (use icon mark only below that threshold)

### Secondary Colors

| Color | Name | Hex | Use |
|---|---|---|---|
| Medium Gray | Brand Gray | `#555555` | Secondary text, "PROJECT" wordmark, body copy |
| White | Background | `#FFFFFF` | Primary background, negative space |

### Icon Color Behavior by Version
- **Light background versions** — Orange bulb outline + orange fill + black interior details (mic, filament)
- **Dark background versions** — Orange icon unchanged, wordmark switches to white
- The orange never changes across versions. It is the fixed anchor of the mark.

### Color Logic
- **Brand Orange** is the energy of the brand — it signals action, attention, and value. Use it where you want eyes to go.
- **Near Black** is the authority of the brand — weight, clarity, confidence. Primary text and structure.
- **Gray** is the support layer — it softens without disappearing. Secondary information, descriptors, and labels.
- **White** is the breathing room — the system should never feel cluttered.

## Typography

### Primary Font — LESARUSS Wordmark
- **Identified as:** Condensed ultra bold sans-serif (likely **Montserrat Black** or **Anton** — confirm from source)
- **Weight:** Black / ExtraBold (900)
- **Style:** All caps, zero letter-spacing
- **Use:** Headlines, primary display text, brand name

### Secondary Font — "PROJECT" / Body
- **Identified as:** Light/thin weight of the same sans-serif family
- **Weight:** Light (300) or Regular (400) with wide letter-spacing applied
- **Use:** Subheadings, labels, supporting text, section titles
- **Letter-spacing:** Wide (+2 to +4 tracking)

### Type Pairing Logic
The pairing works because it uses **weight contrast within a single family** — not two different typefaces. The heavy wordmark commands attention; the light label recedes and supports. Apply this same logic across all brand materials: lead with bold, support with light.

## Design Principles

### 1. Weight Contrast
Use heavy and light together. The brand is not one monotone thing — it is bold where it needs authority and light where it needs to breathe.

### 2. Clarity Over Decoration
The logo is clean and functional. Design across the brand should follow the same standard: no gratuitous decoration, no elements that don't earn their place.

### 3. The Icon Is a System
The bulb + mic icon communicates the brand concept in a single mark. Every visual decision should support that concept — systems that amplify, ideas that become machines.

### 4. Amber Draws the Eye
Gold/amber is used sparingly and purposefully. When it appears, it means something. Do not dilute it by overuse.

### 5. White Space Is a Feature
The system should feel organized and open — not packed. White space is not empty. It is the visual equivalent of structure.
